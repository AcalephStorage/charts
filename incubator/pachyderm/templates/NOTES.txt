1. Download Pachctl:

For OSX:
$ brew tap pachyderm/tap && brew install pachctl

For Linux (64 bit):
$ curl -o /tmp/pachctl.deb -L https://pachyderm.io/pachctl.deb && sudo dpkg -i /tmp/pachctl.deb

2. Once installed start port-forwarding:

export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app=pachd" -o jsonpath="{.items[0].metadata.name}")
kubectl port-forward $POD_NAME 30650:650 -n {{ .Release.Namespace }}

3. In another terminal confirm pachyderm is available:

pachctl version

4. For further information, read the documentation: http://pachyderm.readthedocs.io/en/latest/index.html
